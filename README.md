# Nathan-s-Music-Player

This personal project was made while learning how to create GUIs and utilize audio files in Java. 

## Features
- File Selection 
- Shuffle 
- Previous
- Play/Pause
- Next
- Loop/Unloop
- Scrollable Playlist
- Current Song Display
- Progress Bar
- Timestamp
- Volume Control
- Snail Listening to Music
